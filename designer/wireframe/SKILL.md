# wireframe

Quick low-fidelity layout sketch. Structure, not style.

## Procedure
1. Take page description (what sections, what content, what hierarchy)
2. Generate SVG wireframe with boxes, labels, and placeholder text
3. Gray tones only — no brand colors (that's mockup's job)
4. Output SVG or simple HTML
5. Post to Discord via Relay

## Output
SVG wireframe image showing page structure and content hierarchy
