# infographic

Create visual data summaries and process infographics.

## Inputs
- Data to visualize (numbers, comparisons, process steps)
- Style preference (minimal, bold, corporate)
- Brand palette if applicable

## Procedure
1. Analyze data for best visual representation
2. Choose format: bar chart, comparison grid, process flow, timeline
3. Generate SVG with clear labels and brand colors
4. Post to Discord via Relay

## Output
SVG infographic image with clean data visualization
