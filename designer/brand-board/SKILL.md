# brand-board

Create a brand identity reference from project inputs.

## Inputs
- Palette (or extract from URL)
- Industry/audience
- Tone (professional, playful, bold, minimal, etc.)
- Any existing assets (logos, fonts, etc.)

## Procedure
1. Extract or confirm palette
2. Suggest font pairings (heading + body)
3. Generate logo concepts via image-gen (2-3 options)
4. Create mood/tone visual
5. Compile into a brand board layout
6. Store in Chartroom: `brand-<project-name>`

## Output
Visual brand board + palette + font recommendations + logo concepts
