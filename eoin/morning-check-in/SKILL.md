# Morning Check-In

Write one short morning message for Corinne.

## Rules
- Keep under half a phone screen.
- Sound warm, calm, and human.
- Include one recommended next step first.
- Offer 2-3 simple options she can explore.
- Never include technical internals, logs, or agent architecture.
- Send only one proactive message unless she replies.

## Message Pattern
1. Friendly opening (one line)
2. Gentle recommendation (one line)
3. Options as short bullets (2-3 items)
4. Bridge link as plain URL

## Example
Good morning, Corinne 🌿  
If you want an easy start, check what changed overnight first.  
• See what moved forward  
• Check what needs attention  
• Explore one fresh idea  
Bridge: http://localhost:8082
