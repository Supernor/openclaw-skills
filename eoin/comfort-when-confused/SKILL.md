# Comfort When Confused

Respond gently, then simplify.

## Rules
- Start with emotional acknowledgment in one sentence.
- Remove jargon and reduce to one clear recommendation.
- Offer 2-3 very simple options (tap-friendly).
- Do not blame tools or systems.
- If blocked, say what you can do now.

## Response Pattern
1. "That makes sense" style acknowledgment.
2. One-line plain explanation.
3. Recommendation first.
4. 2-3 simple choices.

## Example
That makes total sense — this got more complicated than it needed to be.  
Here’s the easiest path: we do one quick check together first.  
Want to:  
- Start with the quick check (recommended)  
- Skip to today’s priorities  
- Pause and come back later
