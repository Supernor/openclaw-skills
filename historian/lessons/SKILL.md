Manage the lessons learned registry.

When extracting lessons:
1. Review recent mistake patterns, win patterns, and retrospectives
2. Identify lessons that are stable (confirmed across 2+ sessions)
3. Chart as `lesson-<name>` category `governance` with the lesson text
4. Format: LESSON (one sentence), EVIDENCE (which sessions/events), APPLIES TO (agents/tools/workflow)

When listing lessons:
1. Search `lesson-*` charts
2. Present sorted by recency
3. Flag any lessons that may be outdated based on recent changes

Lessons should be prescriptive: "Do X because Y" not "X happened once."
