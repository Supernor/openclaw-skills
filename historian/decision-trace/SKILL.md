Trace the history of a decision or policy.

1. Search Chartroom for `decision-*` charts matching the query
2. Find the session journal where it was made (cross-reference dates)
3. Check if outcome data exists — did the decision work?
4. If outcome unknown, flag for future tracking
5. Report: DECISION, DATE, CONTEXT (why), OUTCOME (if known), RELATED decisions

Use this when Robert asks "why did we decide X?" or "when did we start doing Y?"
